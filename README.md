# 📊 Mobile Sales Dashboard (Power BI)

## 📌 Project Overview

This project is an interactive Power BI dashboard built to analyze mobile phone sales data.
It provides insights into sales performance, customer behavior, product trends, and payment preferences.

---

## 📂 Dataset

The dataset includes:

* Transaction details
* Mobile brands and models
* Customer demographics
* Payment methods
* Sales data (units & price)

---

## 🛠️ Data Preparation

The following transformations were applied:

* Created a **Date column** from Day, Month, Year
* Generated **Month Name & Month Number** for time analysis
* Standardized **Day Name**
* Created **Total Sales column**
* Cleaned categorical fields (Brand, Payment Method)

---

## 📊 Dashboard Features

### 🔹 KPIs

* Total Sales
* Total Quantity
* Number of Transactions
* Average Price
* Year-over-Year Growth %

### 🔹 Analysis

* 📱 Best-selling mobile models
* 🏷️ Brand comparison (Apple, Samsung, etc.)
* 💳 Payment method analysis
* 📈 Monthly sales trend
* 🌍 Sales by city

---

## 🎛️ Filters

* Brand
* Mobile Model
* Payment Method
